# Costa Terapias — Website Institucional

Site de alta conversão desenvolvido para a Costa Terapias, espaço de relaxamento profundo localizado em Cabo Frio, RJ.

## Sobre o projeto

Página única (single page) com foco em conversão, neurolinguagem sensorial e experiência visual imersiva. Construída inteiramente em HTML, CSS e JavaScript puro, sem dependências externas ou frameworks.

## Funcionalidades

- Trilíngue: português, inglês e espanhol com alternância instantânea
- Formulário de agendamento integrado ao WhatsApp com mensagem pré-preenchida
- Galeria de serviços com 10 tratamentos, cada um com imagem e descrição sensorial
- Seção de experiência completa Day Spa
- Depoimentos, galeria de ambiente e seção de citação
- Navegação fixa com scroll reveal e menu mobile responsivo
- Scroll reveal progressivo nos cards de serviços

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis, Grid e Flexbox
- JavaScript vanilla com IntersectionObserver
- Google Fonts: Cormorant Garamond + Nunito
- Imagens via Unsplash CDN
- Logo embutida em base64 para independência de arquivos externos

## Identidade visual

Paleta baseada na marca oficial Costa Terapias: azul `#2d4a62`, dourado `#b8996e` e creme `#ede8e0`.

## Deploy

Arquivo único `index.html`. Compatível com GitHub Pages, Netlify ou qualquer hospedagem estática.
